Descrizione Software:
WinPy32 permette di compilare online molto semplicemente programmi in Python.
Una volta caricato il file premendo l'apposito pulsante "Scegli file", premendo sul pulsante "INVIA" il software inizierà la compilazione e una volta terminata scaricherà automaticamente il file sul PC.
È inoltre possibile scegliere il tipo di compilazione da fare, se per programmi a riga di comando o per programmi con interfaccia grafica.
Il motore di compilazione usato è PyInstaller con l'estensione PypiWin32.
Il software per motivi di retro-compatibilità compila file .exe solo a 32 bit.
